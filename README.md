# 🛠️ 기술 스택
- Language : `Java`
- Database : `MySQL`
- JDK : `OpenJDK 21`
- API 기능 : `POST / create / diary`, `GET / read / diary`, `GET / read / diaries`, `PUT / update / diary`, `DELETE / delete / diary`
- Library : `Spring data jpa`, `Spring`, `MySQL`, `json-simple`, `Lombok`, `mockito`, `Junit5`, `swagger`
